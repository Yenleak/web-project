Project Description:
TaskFlow is a task management system that helps users 
organize their daily tasks with deadlines, sub-tasks, and 
real-time deadline reminders.


What does the app do?:

- User Authentication: users can register, log in, and log out securely.

- Task Management: users can create, edit, delete, and manage tasks. Each task 
can be linked to a specific category and marked as completed 
or pending.

- Deadline & Reminders: each task can have a specific deadline with date and time. 

- Sub-tasks: users can break down large tasks into smaller sub-tasks. 

- Category Organization: users can create and manage custom categories to group 
related tasks together for better organization.

- Filtering and Search: users can filter tasks by category or status, and search 
tasks by title to find what they need quickly.

Comments: users can add comments to tasks to leave notes or 
additional details.

Technologies:
- Python
- Django
- Django REST Framework
- Simple JWT
- django-cors-headers
- SQLite

Main Features:
- JWT authentication: register, login, logout, profile
- Full CRUD for tasks
- CRUD for categories
- Task filtering by category
- Task filtering by status
- Task search by title
- Comments for tasks
- Objects linked to authenticated user
